# Project Title
capstone-api-starter
## Description of the Project
Ecommerce Api & Web App - In this Project we did Backend work for an Rest API using Java and Spring Boot testing in insomnia for Bugs.
Fixing the core online functionality of product browsing, category management, shopping cart persistance, user profiling, and order Creation. 
the application follows REST & CRUD principles with role based access. It fetures a MySQL database using controllers and Data Access Objects for 
functionallity, stucture and maitnance. Lastly it empahsizes data validation, bug fixing as one would in traditional backend work.

## User Stories

List the user stories that guided the development of your application. Format these stories as: "As a [type of user], I want [some goal] so that [some reason]."

Phase 1 - User Storie

As a shopper, I want to view all product categories so that I can easily browse products by type.
As a shopper, I want to view a single category so that I can see its name and description.

Phase 2 – Products & Bug Fixes User Stories

As a shopper, I want to view all products so that I can browse available items.
As a shopper, I want to filter products by category, price range, or subcategory so that I can quickly find what I want.


Phase 3 – Shopping Cart User Stories

As a logged-in user, I want to view my shopping cart so that I can see all items I plan to purchase.
As a logged-in user, I want to add products to my shopping cart so that I can purchase them later.
As a logged-in user, I want items in my cart to persist after logging out so that my cart is saved between sessions.
As a logged-in user, I want to update the quantity of an item in my cart so that I can adjust my order.
As a logged-in user, I want to remove all items from my cart so that I can start over if needed.

Phase 4 – User Profile (Optional) User Stories

As a registered user, I want a profile to be automatically created when I register so that my personal details are stored.
As a logged-in user, I want to view my profile so that I can see my saved information.
As a logged-in user, I want to update my profile so that my contact details stay current.


Phase 5 – Checkout / Orders (Optional)


User Stories

As a logged-in user, I want to place an order from my shopping cart so that I can complete my purchase.
As a logged-in user, I want my cart to be converted into an order so that all items are recorded permanently.
As a logged-in user, I want my shopping cart cleared after checkout so that it is ready for a new order.
